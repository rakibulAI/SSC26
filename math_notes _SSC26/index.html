অধ্যায় ২: সেট ও ফাংশন

Set & Function | উপসেট, অপারেশন, ভেনচিত্র, ফাংশন

🔷 সেটের সংজ্ঞা ও প্রকারভেদ

সুনির্দিষ্ট বস্তুর সংগ্রহকে সেট বলে। যেমন: A = {1, 2, 3}

প্রকারবিবরণউদাহরণ
সসীমসীমিত সদস্য{1,2,3}
অসীমঅসীম সদস্যℕ
ফাঁকাকোনো সদস্য নেই∅
এককএকটিমাত্র সদস্য{5}
সার্বিকসকল সদস্যের সেটU
⚡ n(A) = সেট A-এর মোট সদস্য সংখ্যা
🔷 উপসেট ও গণনা
  • A ⊆ B → A, B-এর উপসেট
  • A ⊂ B → প্রকৃত উপসেট (A ≠ B)
  • n সদস্যের সেটে উপসেট = 2ⁿ
  • প্রকৃত উপসেট = 2ⁿ − 1
n(A) = 3 → উপসেট = 2³ = 8টি
💡 ফাঁকা সেট (∅) সকল সেটের উপসেট
🔷 সেট অপারেশন
সংযোগ (Union)
A ∪ B = A ও B উভয়ের সকল সদস্য
ছেদ (Intersection)
A ∩ B = উভয়ের সাধারণ সদস্য
পার্থক্য
A − B = A-তে আছে B-তে নেই
পূরক
A' = U − A
🔷 ভেনচিত্র ও মূল সূত্র
n(A∪B) = n(A) + n(B) − n(A∩B)
ডি-মর্গানের সূত্র 🔥
(A∪B)' = A' ∩ B'
(A∩B)' = A' ∪ B'
🔥 পরীক্ষায় প্রতি বছর আসে!
🔷 ফাংশন (Function)

A থেকে B-তে এমন সম্পর্ক যেখানে A-এর প্রতিটি উপাদানের B-তে একটিমাত্র মান।

f: A → B
ধরনবৈশিষ্ট্য
এক-এক (One-One)ভিন্ন ইনপুট → ভিন্ন আউটপুট
উপরি (Onto)সকল codomain ব্যবহৃত
Bijectiveএক-এক এবং উপরি উভয়ই
অভেদ (Identity)f(x) = x
Domain = ইনপুট | Range = আউটপুট | Codomain = B সেট
🔷 গুরুত্বপূর্ণ সেট সূত্র সংকলন
A ∩ ∅ = ∅
A ∪ ∅ = A
A ∩ A' = ∅
A ∪ A' = U
(A')' = A
A ∩ U = A
A ∪ U = U
n(A∪B∪C) = n(A)+n(B)+n(C)−n(A∩B)−n(B∩C)−n(A∩C)+n(A∩B∩C)

অধ্যায় ৩: বীজগাণিতিক রাশি

সূত্র, উৎপাদক বিশ্লেষণ, ভাগশেষ উপপাদ্য, গ.সা.গু ও ল.সা.গু

🔷 মূল সূত্রসমূহ (অবশ্যই মুখস্থ!)
(a+b)² = a²+2ab+b²
(a−b)² = a²−2ab+b²
(a+b)³ = a³+3a²b+3ab²+b³
(a−b)³ = a³−3a²b+3ab²−b³
a²−b² = (a+b)(a−b)
a³+b³ = (a+b)(a²−ab+b²)
a³−b³ = (a−b)(a²+ab+b²)
🔷 পরিচয় সূত্র
a²+b² = (a+b)²−2ab = (a−b)²+2ab
(a+b+c)² = a²+b²+c²+2(ab+bc+ca)
a³+b³+c³−3abc = (a+b+c)(a²+b²+c²−ab−bc−ca)
🔥 a+b+c=0 হলে → a³+b³+c³ = 3abc
(a+b)²+(a−b)² = 2(a²+b²)
(a+b)²−(a−b)² = 4ab
⚡ শেষ দুটি সূত্র পরীক্ষায় বেশি আসে!
🔷 উৎপাদকে বিশ্লেষণ পদ্ধতি
১
সাধারণ উৎপাদক: 2x²+4x = 2x(x+2)
২
সূত্র প্রয়োগ: x²−9 = (x+3)(x−3)
৩
মধ্যপদ বিশ্লেষণ: x²+5x+6 = (x+2)(x+3)
৪
গুচ্ছীভবন: ax+ay+bx+by = (a+b)(x+y)
💡 মধ্যপদ: গুণফল = ধ্রুব পদ, যোগফল = মধ্যপদের সহগ
🔷 ভাগশেষ ও গুণনীয়ক উপপাদ্য
ভাগশেষ উপপাদ্য: f(x)÷(x−a) এর ভাগশেষ = f(a)
গুণনীয়ক উপপাদ্য: f(a)=0 হলে (x−a) উৎপাদক
📌 f(x)=x³−6x²+11x−6 → f(1)=0 → (x−1) উৎপাদক
⚡ প্রথমে ±1, ±2 মান পরীক্ষা করো!
🔷 গ.সা.গু ও ল.সা.গু (বহুপদী রাশির)

১

৩

১
উৎপাদকে বিশ্লেষণ করো
২
সকল উৎপাদকের সবচেয়ে বড় ঘাত নাও
৩

অধ্যায় ৯: ত্রিকোণমিতিক অনুপাত

sin, cos, tan — পরিচয় সূত্র, বিশেষ কোণ, উচ্চতা ও দূরত্ব

🔷 মূল অনুপাত
sin θ = P/H | cos θ = B/H | tan θ = P/B
cosec = H/P | sec = H/B | cot = B/P
⚡ SOH–CAH–TOA মনে রাখো!

cos: উল্টো ক্রমে কমে
🔷 পরিচয় সম্পর্ক (Identity)
sin²θ + cos²θ = 1
1 + tan²θ = sec²θ
1 + cot²θ = cosec²θ
  • sin²θ = 1 − cos²θ
  • tan θ = sin θ / cos θ
  • sin θ · cosec θ = 1
  • cos θ · sec θ = 1
  • tan θ · cot θ = 1
🔷 উন্নতি ও অবনতি কোণ

  • অবনতি কোণ: অনুভূমি থেকে নিচে দেখার কোণ
  • উন্নতি কোণ = অবনতি কোণ (alternate angles)

অধ্যায় ১১: অনুপাত ও সমানুপাত

Ratio & Proportion | Componendo-Dividendo

🔷 অনুপাত (Ratio)
a : b = a/b (b ≠ 0)
ধরনবিবরণ
সরল অনুপাতa:b
ব্যস্ত অনুপাতb:a
যৌগিক অনুপাতa:b ও c:d → ac:bd
বর্গ অনুপাতa²:b²
বর্গমূল অনুপাত√a:√b
🔷 সমানুপাত (Proportion)
a:b = c:d → ad = bc
  • মধ্যসমানুপাতী: b = √(ac)
  • তৃতীয়সমানুপাতী: c = b²/a
  • চতুর্থসমানুপাতী: d = bc/a
b² = ac (মধ্যসমানুপাতীর শর্ত)
🔷 সমানুপাতের ধর্মসমূহ

যদি a/b = c/d হয়, তবে:

বিনিমেয়: b/a = d/c
পর্যায়ক্রমী: a/c = b/d
যোজনী: (a+b)/b = (c+d)/d
বিয়োজনী: (a−b)/b = (c−d)/d
C&D: (a+b)/(a−b) = (c+d)/(c−d)
⚡ Componendo & Dividendo পরীক্ষায় প্রায়ই আসে!
🔷 সমমান অনুপাত
a/b = c/d = e/f = (a+c+e)/(b+d+f)
= (pa+qc+re)/(pb+qd+rf)
📌 x/2 = y/3 = z/5 হলে → x:y:z = 2:3:5

🔷 শ্রেণীবদ্ধ উপাত্তের মধ্যক
মধ্যক = L + {(n/2 − F)/f} × h

🔷 বিস্তারের পরিমাপ
পরিসর = সর্বোচ্চ − সর্বনিম্ন
গড় বিচ্যুতি: MD = Σ|x−x̄| / n
প্রমাণ বিচ্যুতি: σ = √{Σ(x−x̄)²/n}
সহজ: σ = √{(Σx²/n) − (x̄)²}
CV = (σ / x̄) × 100%
⚡ CV কম = উপাত্ত বেশি সামঞ্জস্যপূর্ণ
