# 📚 SSC26 Study Notes (HTML Version)

This repository contains **short study notes for SSC 2026 students**.  
All notes are written in **simple HTML files** so anyone can open them easily on mobile or computer.

Subjects included:
- Physics
- Chemistry
- Math

These notes are designed for **quick revision before exams**.

---

# 🌐 How to View the Notes

You can view the HTML notes in two ways.

## Method 1 — Open Directly (Recommended)

1. Click on any folder:
   - `physics_notes`
   - `chemistry_notes`
   - `math_notes_SSC26`

2. Click the **HTML file**.

3. Press **Download**.

4. Open the file on your phone.

5. The note will open in your **browser**.

---

# 📱 Method 2 — Run on Your Mobile Local Server

If you want a better experience, you can run the files on a **local server**.

### Step 1 — Download the Repository

Download the project as ZIP:

1. Click the green **Code** button.
2. Click **Download ZIP**.
3. Extract the ZIP file on your phone.

---

### Step 2 — Install a Local Server App

Install one of these apps from Play Store:

- **Kiwi Browser**
- **Acode**
- **Spck Editor**
- **Web Server for Chrome**

---

### Step 3 — Open the HTML Files

1. Open the folder you downloaded.
2. Go inside any subject folder.
3. Tap the `.html` file.

Example:
Your browser will open the note like a website.
